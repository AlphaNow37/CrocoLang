# CrocoLang
Croco is a language that compile to python bytecode

# Example
```
#This is a comment
if 1 == 1:
    print("Hello World")  # Hello World
elif 1 == 2:
    pass
a = 5
while a > 0:
    print(a)  # 5 4 3 2 1
    a -= 1
else:
    print("Done")  # Done
for i in range(5):
    print(i)  # 0 1 2 3 4
```
See more examples in the examples folder
